Find the right post from various URL pattern. Especially permalink starts with post_id.

== Description ==

This plugin will find the right post from various URL pattern. Especially permalink starts with /%post_id%-%postname%/ format.

Allows patterns below

1. /%post_id%-<any characters>/
2. /%post_id%/
3. /%postname%/

if post is found, will redirect the latest permalink.
